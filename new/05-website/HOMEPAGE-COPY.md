### Tool 2: SMED Module

**Subheadline:** Reduce each changeover by 50% or more

**Body:**

Sequence optimization reduces *how many* changeovers you do. SMED reduces *how long* each one takes.

Based on Toyota's proven methodology, our SMED module helps you:
- Analyze every step of your changeover
- Identify what can be done while the machine runs
- Track improvement ideas to completion
- Create standard work for consistent execution
- Time actual changeovers against the standard

**Results:** 50-80% reduction per changeover

**Key Features:**
- Step-by-step changeover analysis
- Internal vs external classification
- Improvement tracking with status workflow
- Standard work generator
- Built-in changeover timer
- Performance analytics over time
