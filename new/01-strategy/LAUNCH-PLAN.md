## Target Market

### Primary Users

| Persona | Role | Pain Point | Value Proposition |
|---------|------|------------|-------------------|
| **Production Planner** | Schedules daily/weekly production | Hours in Excel, suboptimal sequences | 20 min planning, optimal sequence |
| **CI Manager** | Leads improvement projects | No systematic SMED tracking | Structured methodology + tracking |
| **Lean Consultant** | Delivers client projects | Needs tools for workshops | Professional software + certification |

### Target Industries

- Packaging & converting
- Food & beverage
- Plastics & injection molding
- Metal fabrication
- Pharmaceutical
- Any discrete manufacturing with changeovers

### Geography

- Primary: Europe (Nordic, DACH, UK)
- Secondary: North America
- Language: English first, localization later

## Tech Stack Summary

| Layer | Technology | Why |
|-------|------------|-----|
| Framework | Electron | Cross-platform, JS ecosystem, easy testing |
| Frontend | React 19 + TypeScript | Modern, component-based, type-safe |
| Styling | Tailwind CSS + shadcn/ui | Rapid development, consistent design |
| Database | SQLite (better-sqlite3) | Local, fast, no server needed |
| Build | Electron Forge | Standard Electron build tool |
| Testing | Playwright + Vitest | E2E + unit testing |
| Licensing | Paddle | SaaS billing, handles EU VAT |
| Updates | electron-updater | Auto-updates |
