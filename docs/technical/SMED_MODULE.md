### 7. Live Link & Smart Feedback (Implemented Phase 2)
The "Live Link" feature creates a virtuous cycle between SMED analysis and the Global Optimizer.

**Workflow:**
1.  **Analyze**: User performs SMED study on a "Color Change".
2.  **Optimize**: Reduces time from 20m -> 15m.
3.  **Publish**: When publishing the Standard, user sees a **Smart Preview**.
    *   *Preview*: "Impact: 20m -> 15m (Color Change)".
    *   *Action*: User explicitly confirms the update.
4.  **Sync**: The system updates the global `ChangeoverAttribute` or `ChangeoverMatrix`.
5.  **Result**: The next optimized schedule automatically uses the new 15m standard.

## Features

### 1. Study Management

Create and manage changeover studies for different product-to-product transitions.

**Capabilities:**
- Create study with name, description, from/to products
- Track study status (draft, analyzing, improving, standardized)
- Record baseline time and target time
- Attach notes and observations
- Archive completed studies

### 2. Step Analysis

Break down changeovers into individual steps and classify them.

**Step Properties:**
- Sequence number
- Description
- Duration (minutes:seconds)
- Category (preparation, removal, installation, adjustment, cleanup)
- Type (internal vs external)
- Notes and observations
- Video timestamp reference (optional)

**Classification:**
- **Internal**: Must be done while machine is stopped
- **External**: Can be done while machine is running

### 3. Improvement Tracking

Capture and track improvement ideas through implementation.

**Improvement Types:**
- Convert internal to external
- Streamline internal operations
- Parallelize operations
- Eliminate unnecessary steps
- Standardize components
- Use quick-release mechanisms

**Status Workflow:**
```
IDEA â†’ PLANNED â†’ IN PROGRESS â†’ IMPLEMENTED â†’ VERIFIED
```

**Tracking Fields:**
- Description
- Estimated time savings
- Cost to implement
- Assigned to
- Due date
- Actual savings achieved

### 4. Standard Work

Generate standard operating procedures from optimized studies.

**Standard Contents:**
- Step-by-step procedure
- Standard time per step
- Total standard time
- Required tools list
- Safety precautions
- Visual aids/photos
- Revision history

**Output Formats:**
- Printable checklist (PDF)
- On-screen reference
- Timer mode (guided execution)

### 5. Changeover Timer

Time actual changeovers and compare to standard.

**Timer Features:**
- Start/stop with keyboard shortcut
- Step-by-step timing (optional)
- Visual progress vs standard time
- Record notes during changeover
- Automatic log creation

### 6. Performance Analytics

Track changeover performance over time.

**Metrics:**
- Average changeover time (by type)
- Trend over time
- Variance from standard
- Best/worst performances
- Improvement progress
