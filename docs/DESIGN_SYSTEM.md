# UX-06: Design System

**Colors, Typography, Spacing, and Components**

---

## Purpose

This document defines ChangeoverOptimizer's visual language. Every color, font size, and spacing value is specified here. Consistency = trust = professional tool.

---

## Design Philosophy

> **"Industrial precision, not startup playfulness."**

ChangeoverOptimizer serves manufacturing professionals. The visual design should feel:

| Feel | Not |
|------|-----|
| Professional | Playful |
| Trustworthy | Trendy |
| Efficient | Decorative |
| Clear | Clever |
| Calm | Exciting |

**Reference aesthetics:** Enterprise software (SAP, Siemens), industrial dashboards, engineering tools.

---

## Responsive Considerations

ChangeoverOptimizer is a desktop app, but should handle window resizing gracefully.

### Breakpoints

| Name | Width | Behavior |
|------|-------|----------|
| Minimum | 800px | Fixed min-width, horizontal scroll below |
| Narrow | 800-1000px | Compact layout, reduce padding |
| Normal | 1000-1400px | Default layout |
| Wide | 1400px+ | Max content width, center |

### Adaptation Rules

| Element | Narrow | Normal | Wide |
|---------|--------|--------|------|
| Screen padding | 24px | 32px | 48px |
| Card grid | 1 column | 2 columns | 3 columns |
| Table | Horizontal scroll | Fit width | Fit width |
| Modal | 90% width | 500px | 600px |
